func FlatMap ¶ 

func FlatMap[K1, K2 comparable, V1, V2 any](fn func(k K1, a V1) map[K2]V2) func(a map[K1]V1) map[K2]V2

Apply a function to each element of map, and flatten the nested map

▹ Example

▾ Example

Code:

animalList := map[string]int{
    "cat":  1,
    "dog":  2,
    "fish": 3,
}

flipKV := FlatMap(func(k string, v int) map[int]string {
    return map[int]string{
        v: k,
    }
})
fmt.Println(
    flipKV(animalList),
)

Output:

map[1:cat 2:dog 3:fish]
